YouGrabber - Premium YouTube Downloader PHP Script

  • created: 02/01/2017
  • by: oraksoft

Introduction

Welcome to YouGrabber. A premium "YouTube Downloader" PHP script that works. Really easy to install, light-weight and the best thing about it - it doesn't require any sort of additional softwares installed on your server. Pure PHP script, that will grab info about YouTube's video and let you download it in all of the available formats.

Requirements

Well, this script will require at least these things:

  • Apache Server
  • PHP 5.3 (+) installed

How to install

Hey, I promise you that this will be one of the easiest PHP script installations you've ever done.

Just exctract the archive you've downloaded from CodeCanyon. Inside you'll find a folder called "yougrabber". Just move it to your server directory and enter it - you're done. The script should start working and now you can go ahead and make some customization or leave it as is and enjoy it.

If something went wrong and you can't install it correctly, don't hesitate and contact us right away. We will be more than happy to help you.

How to change the logo & etc?

Alright, you may want to change the background, the logo or some other images used.

They are in /yougrabber/application/views/TemplateFrontEnd/images

Go ahead, check the images we've used, change them and make them match your taste (if they already don't) :)

How to change social links?

Go to /yougrabber/application/controllers/main.php

Go to line 8 & 9 and change Envato's twitter and facebook addresses with your twitter and facebook page address.

Customizing the HTML

Well, if you dare, if you have HTML & CSS skills and want to make it more awesome, go ahead.

Here is where the HTML file is located:

HTML: /yougrabber/application/views/site-front-end.php

CSS: /yougrabber/application/views/TemplateFrontEnd/css/style.css
